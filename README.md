# mahjong

## Application for simulating, analyzing, and playing [Three-Player mahjong](http://www.japanese-mahjong.com/3pmjintro.html).

Usage
-----

Start the application with

    node app.js

Then visit [http://localhost:3000](http://localhost:3000).
